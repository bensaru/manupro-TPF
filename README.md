# TPF Piscinas PR Website

A premium one-page marketing website for TPF Piscinas PR, focused on custom pool construction, jacuzzi installation, and outdoor remodeling in Puerto Rico.

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure

- `index.html` - Main website structure and content
- `styles.css` - Full visual system, responsive layout, and component styles
- `script.js` - Mobile menu, WhatsApp form behavior, scroll interactions
- `assets/images/` - Project and gallery images

## Features

- Premium hero section with strong branding
- Services, projects, process, and FAQ sections
- Social showcase with links to Instagram and Facebook
- Contact section with direct call/WhatsApp/Instagram actions
- Contact form that opens WhatsApp with pre-filled message
- Responsive layout for desktop, tablet, and mobile

## Run Locally

This is a static site. No build process is required.

1. Open `index.html` directly in a browser, or
2. Use a local server (recommended), for example:

```bash
python -m http.server 5500
```

Then visit `http://localhost:5500`.

## Customization

- Update business copy in `index.html`
- Update colors/typography/layout in `styles.css`
- Update WhatsApp target number and message format in `script.js`
- Replace images in `assets/images/` and keep the same file names (or update paths in `index.html`)
